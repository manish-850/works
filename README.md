# WØRKS Studio UI Clone

A UI clone of the award‑winning **Works Studio** website, recreated for
practice and learning.

## Overview

This project is a front-end UI clone inspired by the Works.studio
website featured on Awwwards.\
It focuses purely on visual replication and motion experience using
modern web technologies.

## Features

-   Smooth scrolling using **Locomotive Scroll**
-   Loader animation with video integration
-   Minimalistic navigation bar
-   Typography and spacing inspired by original Works.studio aesthetics
-   Custom animations and transitions\
-   Responsive-ready structure (responsiveness not fully implemented
    yet)

## Tech Stack

-   HTML5
-   CSS3
-   JavaScript
-   Locomotive Scroll
-   Remix Icon CDN

## Project Structure

    |-- index.html
    |-- style.css
    |-- script.js
    |-- /img
        |-- video.mp4

## How to Run

1.  Clone or download the project folder\
2.  Open **index.html** in your browser\
3.  Ensure `video.mp4` is placed correctly inside `/img/`\
4.  Experience the UI clone
